The full stack, component by component

1. Twilio — phone & SMS layer (~$20–50/mo)

Provisions a business phone number. Handles call forwarding (when the main number is unanswered, calls route to Twilio). Detects missed calls and fires a webhook to n8n. Sends and receives SMS. Must be 10DLC registered for business texting — this is a real compliance step that takes 2–4 weeks to process. Don't skip it.

2. n8n — workflow orchestration (free self-hosted / $20/mo cloud)

The nerve center. Receives the Twilio webhook → extracts caller info → calls the Claude API → sends SMS back via Twilio → loops for each reply in the conversation → detects when the conversation is complete → triggers the output step. All visually built, no full application code required.

3. Claude API — the AI brain (~$0.003–0.015 per conversation)

Receives the conversation history and the system prompt on every message. Returns the next AI response. The system prompt is the product differentiation — it encodes the business's services, qualifying questions, tone, and rules. At average SMS conversation length (~1,500 tokens), cost per conversation is roughly half a cent.

4. CRM connector — the output layer (varies)

When the conversation ends, n8n formats the intake summary and pushes it to the business's job management software. Jobber, HousecallPro, and ServiceTitan all have APIs. Clio is the standard for legal. QuickBooks for accounting. This is where MCP connectors shine — a standardized interface instead of custom API code for each system.

5. Notification layer — alerting the owner (near-zero cost)

Simultaneously: sends formatted email summary (SendGrid, ~$0.001/email), sends Slack/Teams notification with urgency flag, optionally texts a Calendly booking link to the caller. All triggered by the same n8n workflow completion step.

Total infrastructure cost: ~$85/month to serve 10 customers paying $297/month = $2,970 revenue. Gross margin before your time: ~97%. The business model works.

The key architectural decisions

Why n8n over Make or Zapier?
Per-task pricing at Zapier would cost $0.05–0.10 per SMS sent. At 50 messages/day across 10 customers, that's $150–300/month just in Zapier costs. n8n's flat pricing makes unit economics work at scale.
Why Claude over GPT-4o?
Claude follows multi-constraint system prompts more reliably — critical when you're encoding business rules, compliance requirements, and conversation flow all in one prompt. Also: Anthropic's terms are clearer for commercial use cases.
Why SMS over voice AI?
Voice AI (like Sameday) is more impressive but more expensive and has higher fallout rates — people hang up on robot voices. SMS has 98% open rates and people are comfortable qualifying via text. Start with SMS, add voice as an upsell.
Why not build it all custom?
Custom code means custom maintenance. n8n + Twilio + Claude API gives you a maintainable, upgradeable system where each component can be swapped independently. If Claude releases a better model, you update one API call.
